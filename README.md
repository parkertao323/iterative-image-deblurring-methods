# iterative-image-deblurring-methods
This repository includes MATLAB implementations of three iterative methods for image deblurring:
- Cimmino's diagonal method
- Successive over-relaxation method (SOR)
- Steepest descent method

This project studies how these methods can be used to solve image restoration problems of blur models given by convolution.

# files
This repository includes MATLAB codes for each method, and the image data file for deblurring test.
- `Cimmino_Diagonal_Algorithm`
- `SOR_Method`
- `Steepest Descent Method`

# Aim
The main purpose is to compre these 3 iterative methods regarding restoration results in image deblurring problems.
